[[now1]]
===== now(): DateTime

Returns a link:dataweave-types#dates[(Datetime)] object with the current date and time.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: now(),
  b: now().day,
  c: now().minutes
}
----

.Output
[source,json,linenums]
----
{
  "a": "2015-12-04T18:15:04.091Z",
  "b": 4,
  "c": 15
}
----

[TIP]
See link:dataweave-selectors[DataWeave Selectors] for a list of possible selectors to use here.
