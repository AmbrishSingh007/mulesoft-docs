[[isdecimal1]]
== isDecimal(Number): Boolean

Returns `true` if if receives a number that has any decimals in it.

.Transform
[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{
  decimal: isDecimal(1.1),
  integer: isDecimal(1)
}
----

.Output
[source,Json,linenums]
----
  {
    "decimal": true,
    "integer": false
  }
----
